# Haircut-Spa-Booking
A website developed for a unisex Salon using ASP .net MVC in Visual Studio which serves the purpose of saving the customers' precious time by providing them a booking platform for haircut and spa services so that they can view and book their appointment and choose their service in advance. This could avoid crowds at salon and customer-friendly.
